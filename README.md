Gamemodes
=========

This repository contains all relevant configuration files for every gamemode.

**Directory structure**
```
├───gamemode1
│   ├───...
|   ├───mode.json
│   └───Dockerfile
├───gamemode2
|   ├───...
|   ├───mode.json
│   └───Dockerfile
├───gamemode3
|   ├───...
|   ├───mode.json
│   └───Dockerfile
└───gamemode4
    ├───...
    ├───mode.json
    └───Dockerfile
```
Every directory contains configuration for each gamemode. The Dockerfile is used to build the final image.
